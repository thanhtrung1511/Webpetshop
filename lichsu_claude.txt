Tech Stack đề xuất:
- Frontend: Next.js 14 (App Router) + Tailwind CSS
- Backend: Node.js + Express (hoặc Next.js API routes)
- Database: PostgreSQL + Prisma ORM
- Auth: Clerk
- Ảnh sản phẩm: Cloudinary
- Deploy: Vercel (frontend) + Railway (backend/db)

Giai đoạn 1 — Setup dự án (1-2 tuần):
- Tạo dự án Next.js với TypeScript + Tailwind
- Cấu trúc thư mục: app/(shop), app/(booking), app/(auth), admin/, api/
- components/ui, components/shop, components/booking
- lib/prisma.ts, prisma/schema.prisma

Giai đoạn 2 — Tính năng Shop (2-3 tuần):
- Database schema Product: id, name, description, price, images, category (FOOD/CLOTHING/ACCESSORIES/MEDICINE), stock
- Order model với OrderItem, status (PENDING...)

Giai đoạn 3 — Tính năng Đặt Lịch (2 tuần):
- Service model: name (Tắm, Cắt tỉa, Spa), duration, price
- Booking model: userId, serviceId, petName, petType, petSize, date, status, notes
- Logic chọn giờ: calendar hiển thị slot trống theo khung giờ

Giai đoạn 4 — Hoàn thiện & Deploy:
- Tích hợp thanh toán VNPAY/MoMo
- Email xác nhận lịch hẹn (Resend/NodeMailer)
- Trang Admin quản lý đơn hàng, lịch hẹn
- SEO: metadata, sitemap, og:image
- Deploy lên Vercel
